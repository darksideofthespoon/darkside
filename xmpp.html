XMPP Server

We have setup an XMPP sever for those that want to still use OTR now that Google is killing it off. You will need to request an account (using a legit email address), directions are below.

Requesting an account
1. send an email to Info with the subject "Requesting an xmpp account username xxxxxx". Be sure to replace xxxxxx with an actual use name.
2. Wait for an email to confirm account /Â  exchange GPG Keys (could be 24-72 hours)
3. Then 24-72 hours after that you can expect the password sent to you via GPG Encryption

Connection settings
1. Install the DarkSide Root CA from here
2. Username is xxxxxx@darkside.chat
3. Password you will receive via GPG
4. Server URL is "chat.darksideofthespoon.net"
5. Require SSL/TLS
6. If you have a socks5 proxy we encourage you to use it. The admins will be the only ones able to see the IP you connect from, but for maximum privacy we encourage some kind of proxy.
7. We would also encourage you to setup an OTR key for end-to-end encryption, above and beyond the SSL / TLS cert.

Info
This is just a basic Prosdoy xmpp server with some tweaks to the ssl ciphers for better security. We also cannot reset your password, keep a copy in Keepass for safe keeping. Also at this time, there is no good way to let users change their password. We will update that portion as it becomes available. That being said, if you have an OTR fingerprint setup anyone tat has chatted with you can easily tell if it is really you.
